*also known as the op-amp* is component that can compare 2 voltages and limits the output between an external voltage source

the model we will study will be the $\mu A741$ *(the one in the image is 741CN but they are exactly the same, just a manufacturing difference)*
it has 8 terminals as follows:

<center> <img src = "IMAGES/circuit2/op-amp.png"> </center>
both the OFFSET NULL terminals are no use for this study



from the following equation:
$$
V_o = \begin{cases} 
-V_{CC} & A(v_p - v_n) < -V_{CC}, \\[5px]
A(v_p - v_n) & -V_{CC} < A(v_p - v_n) < +V_{CC} \\[5px]
+V_{CC} & A(v_p - v_n) > +V_{CC}
\end{cases}
$$

where $v_p$ and $v_n$ are the non-inverting and inverting inputs respectively 

within very small differences of voltage input, the op-amp behaves as a linear device, but when the difference gets bigger, it **saturates** to the corresponding source voltage

# ideal op-amp
note that the output within the linear region is the input multiplied by a value

this value $A$ is called **gain**, in an ideal op-amp it is so big it is considered $\infty$

leads to the op-amp only outputting either $-V_\text{cc}$ or $V_\text{cc}$ *without any other component connected*
## characteristics of the ideal op-amp 
- $V_p = V_n$
- input resistance = $\infty$
- $i_p = i_n = 0$
- output voltage $v_o$ is always between $-V_\text{cc}$ and $V_\text{cc}$


this was for an op-amp with no other component, now will see what happens when connecting resistances

# The inverting-Amplifier Circuit
![[Pasted image 20260222134212.png]]
An **Inverting Operational Amplifier** is basically fixed-gain voltage amplifier circuit configuration that amplifies an input signal while “*inverting*” its output signal by 180 degrees compared to its input.

That means a positive input signal produces a *negative* output voltage signal and vice-versa. In other words, the output is 180 out-of-phase with the input, producing “inversion”.

## Equations
$$
\begin{array}{c}
\because \quad i_s +i_f = i_n \\[5px]
\because \quad i_s = \dfrac{v_s}{R_s} \;, \quad i_f = \dfrac{v_f}{R_f} \;, \quad i_n = 0 \\[7px]
\therefore v_0 = -\dfrac{R_f}{R_s}v_s
\end{array}
$$

from this relation, we can also express the **gain** $A$ as:
$$
A = -\dfrac{R_f}{R_s} \quad\rightarrow\quad v_o = Av_s
$$


# Summing-amplifier circuit
![[Pasted image 20260222165757.png]]
based on the **Inverting Operational Amplifier** configuration, it combines multiple input voltages into one single output which is proportional to the weighted sum of all the inputs.

for this case, $v_o$ can be expressed as:

$$
v_o = - R_f \left( \dfrac{v_a}{R_a} + \dfrac{v_b}{R_b} + \dfrac{v_c}{R_c} \right)
$$
## special cases
### if $R_a = R_b = R_c = R_s$

$$
v_o = - \frac{R_f}{R_s} \left( v_a + v_b + v_c \right)
$$
### if $R_f = R_s$

$$
v_o = - \left( v_a + v_b + v_c \right)
$$

note that the summing-amplifier doesn't necessarily take 3 inputs, it can take from 2 or more 

# Noninverting-amplifier circuit
![[Pasted image 20260222171004.png]]

same idea as the inverting-amplifier but the output is *in phase* with the input (same sign)

$$
\begin{array}{l}
v_o &= \dfrac{R_s+R_f}{R_f}v_g \\[5px]
&= \left( 1+ \dfrac{R_f}{R_s} \right)v_s
\end{array}
$$

## Note
for our case of an ideal op-amp, the resistance connected to the input source $V_g$ doesn't affect the output. it is connected for the case of real world use

# Voltage Follower
![[Pasted image 20260222172035.png]]
متذكرتش في المرجع ومفيش عليها كلام كتير في ملف الشرح, بس الدكتور أتكلم عنها بردو

assume 2 circuits that you want to connect, and one outputs a voltage 5v for example
if you connect the other circuit to it, it's resistance will combine with the main circuit resistance **draining power** and **dropping the voltage output** decreasing the 5v
if you connect a voltage follower in between, you avoid the other circuit draining power and the 5v remain constant

# Difference-amplifier Circuit
![[Pasted image 20260228225928.png]]
note that the second input in all of the previous states was always ground.

in this configuration we have 2 inputs and the output is proportional to the *difference* between them

$$
v_o = \left( \dfrac{R_a + R_b}{R_a} \right) \left( \dfrac{R_d}{R_d + R_c} \right)v_b - \dfrac{R_a}{R_b}v_a
$$

$$
\text{if: } \dfrac{R_a}{R_b} = \dfrac{R_c}{R_d}
$$

$$
v_o = \dfrac{R_a}{R_b}(v_b - v_a)
$$
